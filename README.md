# scanbot
It's a test nodejs/mongodb/neo4j/reatcjs app that scan internet and build graph of cross links between sites.



Pre-requirements
=======================================
1. Install nodejs + npm: https://nodejs.org/en/



Configuration
=======================================
`cd scanbot`

`npm update`



How to run
=======================================
`cd scanbot`

`npm start`
